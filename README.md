# My Implementation of FriendlyChat

This repository contains code developed along the FriendlyChat project in the [Firebase in a Weekend: Android by Google](https://www.udacity.com/course/firebase-in-a-weekend-by-google-android--ud0352) Udacity course.

This code was built upon the code available at https://github.com/udacity/and-nd-firebase

## Overview

FriendlyChat is an app that allows users to send and receive text and photos in realtime across platforms.

## Setup

Setup requires creating a Firebase project. See https://firebase.google.com/ for more information.
